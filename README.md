## üßê About <a name = "about"></a>

Gr√ºtteChat is a web chat app. Messages are saved in a SQL database. New messages are fetched from the database using AJAX. Messages as well as user passwords are encrypted using the Python cryptography library. But the key is stored on the same machine as the database so thats not really secure. To make the chat more interesting, an AI chat bot is included. The chat bot is built using the openAI GPT-3.5 API. A special bot personality can be used if the signed in Gr√ºtteChat user has bought an upgraded version of Gr√ºtteChat - Gr√ºtteChat PLUS. Currently, the PayPal rest API is used.
