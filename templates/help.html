What is GrütteChat?


GrütteChat is a free, open-source, chat application. It is built with Python, Flask, MySQL, and Django.
Chat messages as well as user information is stored in a MySQL database.
The message content and the user passwords are encrypted before being stored in the database.
GrütteChat also has a state of the art chat-bot called MyAI built in.

How can I use GrütteChat?


Firstly, you need to create an account. You can do this by clicking on the "Sign Up" button on the landing page.
You'll then be asked to enter a username, your email address, and a password.
After that, you'll receive a six digit verification code via email. You'll need to enter this code on the next page.
Once you've done that, you'll be able to log in and use GrütteChat.

What is MyAI


MyAI is a state of the art chat-bot that is built into GrütteChat.
You can access MyAI by clicking on the "MyAI" button on the home page.
As a GrütteChat PLUS user, you'll be able to unlock all MyAI personalities.
Currently, there are 5 additional MyAI personalities available.
You can change your MyAI personality on the settings page

What features does GrütteChat offer?


GrütteChat currently offers the following features:
• Sending and receiving messages to and from other GrütteChat users
• Deleting chats
• Chatting with a state of the art MyAI chat-bot
• Changing your email and password

What is GrütteChat PLUS


GrütteChat PLUS is a premium version of GrütteChat.
It offers the following features:
• Unlock all MyAI personalities
• 4x longer MyAI conversations
• Up to 10x longer MyAI replies
• (Coming soon) Remove ads
• Access to premium chats
• Access to all upcoming features
Additionally, you'll be supporting the developer by purchasing GrütteChat PLUS.

What are premium chats?


Some users have created premium chats. These chats are only accessible to GrütteChat PLUS users.
So to chat with these users, you'll need to purchase GrütteChat PLUS.
But dont worry, you can still chat with all regular users.
This feature is currently in beta.

How does tipping work?


You can tip premium chat users by clicking the icon top left of the chat.
You can then select an amount between 1€ to 10€.
The user you tipped will be notified about your tip.
Unfortunately, tips cannot be refunded.

How can I create a premium chat?


You can't. Premium chats are only available to selected users.

How can I change my password?


You can change your password by clicking on the "Change Password" button on the settings page

How can I change my email address?


You can change your email address by clicking on the "Change Email" button on the settings page

How can I delete my account?


You can delete your account by clicking on the "Delete Account" button on the settings page
